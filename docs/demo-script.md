# AgentVault Demo Script

## One-line Pitch

AgentVault is safe treasury execution for AI agents on Arbitrum.

## Demo Story

NovaPay is a small startup that wants an AI finance agent to help with payments, runway, and vendor operations without giving the agent unrestricted wallet control.

## Judge Flow

1. Show the dashboard balance, runway, and active policy.
2. Open the agent queue.
3. Approve the CloudHost invoice because it is under policy and the vendor is allowlisted.
4. Show the blocked transfer to an unknown recipient.
5. Explain that the AI can propose actions, but smart contracts enforce the rules.
6. Close with the Arbitrum angle: cheap execution, frequent audit logs, and a natural home for agent-operated startups.

## Closing Line

AI agents are becoming operators. AgentVault gives them financial permissions without giving them unlimited control.
